    Sistema de Controle de Estoque

-> Sobre o projeto

Desenvolvi este projeto para praticar controle de estoque utilizando banco de dados relacional, simulando entradas e saídas de produtos.

-> Tecnologias utilizadas

* MySQL / MariaDB (XAMPP)
* SQL

-> Estrutura do banco

* produtos
* fornecedores
* entradas_estoque
* saidas_estoque

-> Funcionalidades

* Cadastro de produtos e fornecedores
* Registro de entrada de mercadorias
* Registro de saída de produtos
* Controle de estoque atualizado
* Consulta com cálculo de saldo

-> O que aprendi

* Modelagem de banco de dados
* Relacionamento entre tabelas
* Uso de JOIN
* Uso de funções agregadas (SUM)
* Controle de dados com lógica de negócio

-> Objetivo

Projeto desenvolvido para portfólio com foco em oportunidades na área de tecnologia.
